 * Os arquivos nn_1.py, nn_2.py e nn_3.py sao arquivos identicos somente mudando a topologia implementada;
 * Quando executados, um diretorio correspondente eh gerada dentro do diretorio "models", contendo a rede
neural treinada;
 * O arquivo execute_nn.py abre uma das redes neurais e pede para o usuario digitar um valor entre 0 e 60;
 * Se o valor nao estiver nesse range, o programa termina;
 * Para mudar qual rede neural deve ser escolhida, alterar a string da linha 13 do arquivo com o nome do
diretorio da rede neural desejada.
